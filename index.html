Dice Game

To start pick which game you would like to play and the amount of money you would like to start with and you will be given instruction on the game rules

High Number

  • In high number place a bet and whoever rolls the higher number wins the pot.

Craps

  • In Craps if you roll either a 7 or an 11 on the first roll you win.
  • If you roll a 2, 3, or 12 on the first roll you lose.
  • If you roll any other number besides that, the number you rolled becomes the "point" and you must roll that number again before rolling a 7.
